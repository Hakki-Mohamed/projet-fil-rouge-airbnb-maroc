# Projet Fil Rouge – Analyse Airbnb Espagne

## Objectif

Comparer le marché Airbnb entre Madrid et Barcelona
à travers l’analyse des prix, types de logements,
disponibilité et segmentation.

## Sources de données

- Inside Airbnb
- Villes étudiées : Madrid, Barcelona
- Fichiers : listings, calendar, reviews, neighbourhoods

## Stack technique

- Python
- Pandas
- Numpy
- Seaborn / Matplotlib
- Jupyter Notebook

## Structure

- data/raw
- data/processed
- notebooks
- dashboards (à venir)
- docs

## Étapes réalisées

- Fusion des listings
- Nettoyage des valeurs manquantes
- Suppression des outliers (99e percentile)
- Analyse descriptive
- Comparaison par ville
- Analyse par type de logement
